Background Graphic Slideshow

The following static HTML will exist on the page.

.bg_slideshow_data will be set to {display:none;}

JS will use this HTML as its data source for generating a slideshow.

JS will create the following HTML for the slideshow:

  <div class="bg_slideshow_slides">
    (for EACH slide that exists in the static HTML)
    <div class="slide_1">
      <a class="whatever" href="#"><strong>First Line!</strong><br />Second Smaller Line</a>
    </div>
    <div class="slide_2">
      <a class="whatever" href="#"><strong>First Line!</strong><br />Second Smaller Line</a>
    </div>
    ...etc, etc, etc.
  </div>
  <div class="bg_slideshow_controls">
    <ul>
      <li><a href="#">previous</a></li>
      <li><a href="#">next</a></li>
    </ul>
  </div>

REQUIREMENTS

  1. The <img>'s need to be pulled by JS and turned into CSS background-images so that a narrow browser window does not trigger a horizontal scrollbar.
  2. JS will also grab the entire HTML of the <a>'s and just place them into the same slide <DIV>. CSS will handle positioning and styling the captions
  3. The slideshow will have the following settings:
    • autoplay on page load
    • crossfade between slides for .5seconds
    • delay between each slide for 3.5 seconds
  4. Details about the machine
    • the "visible" slide will have the highest z-index; the script will cycle the z-indices and .animate the opacity between the two topmost slides to provide the crossfade to the next higher-z-index slide.
    • JS will reorder the slides in a loop. Taking the next slide, giving it z-index:1, putting it on top of the stack of HTML, and then animating the opacity to 100% when the delay is finished.
    • after a new slide is revealed at the top of the stack, the other / lower slides have the following resets: z-index: 0, opacity: 0.
    • .slide height will be determined by the img.height(); and will be added as CSS
    • the class of the <img> will be given to the .slide to provide the ability for manually setting the bg-color
  5. Previous & Next control buttons will navigate forwards and backwards through the slideshow.
  6. Clicking Previous or Next will pause the Autoplay / timer for 10 seconds or so?
